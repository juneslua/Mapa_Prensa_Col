# Infografia

![Map](img/InteractiveMap.gif)

Infographic develop for an online publication back in 2019. The information was collected from the [FLIP "Fundación para la Libertad de Prensa"](https://flip.org.co/index.php), related to the threats face by journalist related to their jobs, since 2006 up to 2019. An SVG map was use to allow for an interactive navegation of the data. 


_Supports both desktop and mobile._
